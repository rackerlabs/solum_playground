Deploy an Application

To deploy an application, you need to provide following information. name of the application, languagepack, application code repository(github url), port etc... For more information refer to Openstack Docs.

Application Parameters

Optionally you can pass parameters to application, for example database connection string, username/password etc.. These parameters will be available as environment variables to your application. Please make sure you are providing correct parameter names which translates to environment variable name.
